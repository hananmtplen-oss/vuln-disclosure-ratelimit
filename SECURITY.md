| Version | Supported |

## Reporting a Vulnerability

Please report security vulnerabilities via GitHub Security Advisories (Private vulnerability reporting).

- **Critical bugs**: $5000 bounty
- **High severity**: $2000 bounty
- **Medium**: $500
- **Low**: $50

We aim to respond within 24 hours and patch within 90 days.
