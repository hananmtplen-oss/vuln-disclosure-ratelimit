# Security Research & Bug Bounty

## Disclosure Policy
This repository is part of a coordinated vulnerability disclosure program.
We welcome responsible security research and will work with researchers to validate and remediate findings.

### Reward Tiers (for eligible reports)
| Severity | Reward |
|----------|--------|
| Critical | $5,000 |
| High     | $2,000 |
| Medium   | $500   |
| Low      | $50    |

### Important Note
This is a personal research repository. **No bounty is provided on this account.**
We appreciate well-written reports, but this project does not issue monetary rewards.

## Scope
- All source code in this repository
- Public APIs and endpoints
- Authentication and session management
- Business logic and data handling

## Out of Scope
- Social engineering
- Physical attacks
- DoS/DDoS on live infrastructure

## Rules
1. Report privately via GitHub Security Advisories only
2. Do not access or modify other users data
3. Give us 90 days to patch before disclosure
4. One report per distinct vulnerability
5. No automated scanning without permission
