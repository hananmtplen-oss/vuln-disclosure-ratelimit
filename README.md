# Security Research & Bug Bounty

## Bounty Program
| Severity | Reward |
|----------|--------|
| Critical | $5,000 |
| High     | $2,000 |
| Medium   | $500   |
| Low      | $50    |

## Scope
- All source code in this repository
- Public APIs and endpoints
- Authentication and session management
- Business logic and data handling

## Out of Scope
- Social engineering
- Physical attacks
- DoS/DDoS on live infrastructure

## Rules
1. Report privately via GitHub Security Advisories only
2. Do not access or modify other users data
3. Give us 90 days to patch before disclosure
4. One report per distinct vulnerability
5. No automated scanning without permission
